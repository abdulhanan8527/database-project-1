* ```npm install express```
* ```npm --save-dev nodemon dotenv```

* ```npm i passport passport-local express-session express-flash```
* ```npm install mongodb```
* ```npm install mongoose```
* ```npm install body-parser --save```
* ```npm install passport-local-mongoose --save```
